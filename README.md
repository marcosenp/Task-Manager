# 📝 Gerenciador de Tarefas 

Um gerenciador de tarefas simples, desenvolvido em C# para execução via console. Permite adicionar, listar, editar e excluir tarefas.

## 🚀 Funcionalidades

- Adicionar tarefa
- Excluir tarefa
- Editar descrição
- Listar tarefas
- Sair do sistema

## 🛠 Tecnologias

C#

.NET Console

Git Flow

Versionamento Semântico
